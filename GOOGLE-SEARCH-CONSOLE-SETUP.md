## 📊 Check Your Current Ranking in Search Console

1. Go to: https://search.google.com/search-console
2. Click **Performance** → **Search Results**
3. Look at these metrics:

| Metric | What It Tells You |
|--------|-------------------|
| **Total Impressions** | How often Google shows your site |
| **Total Clicks** | How many people clicked |
| **Average CTR** | Click-through rate (aim for 3%+) |
| **Average Position** | Your ranking (1 = top, 100 = page 10) |

4. Click **Queries** tab to see keywords you rank for
5. Click **Pages** tab to see which pages get traffic

## Common Issues to Watch For

### 1. "Discovered - currently not indexed"
- **Cause:** Google found the URL but hasn't crawled it yet
- **Fix:** Request indexing manually via URL Inspection

### 2. "Crawled - currently not indexed"
- **Cause:** Google crawled but didn't find it valuable enough
- **Fix:** Improve content quality, add more unique content

### 3. "Duplicate without user-selected canonical"
- **Cause:** Multiple URLs with same content
- **Fix:** Already addressed with unique page content

### 4. "Page with redirect"
- **Cause:** Page redirects somewhere else
- **Fix:** Check redirects in vercel.json
